# ireneuszzasadni.com

Personal website of Ireneusz Zasadni — IREB-certified Business Analyst and freelance consultant.

Static site hosted on **GitHub Pages**, available at [www.ireneuszzasadni.com](https://www.ireneuszzasadni.com/).

## Structure

- `index.html` — home page (hero, services, process, portfolio band, contact CTA)
- `projects.html` — selected project case studies
- `contact.html` — contact form (powered by [FormSubmit](https://formsubmit.co/)) and LinkedIn link
- `thanks.html` — confirmation page after form submission
- `404.html` — not-found page
- `CNAME` — custom domain configuration for GitHub Pages
- `photo.jpg`, `logo-*.png`, `favicon.svg` — assets

No build step — plain HTML/CSS. Edit files and push to `main`; GitHub Pages deploys automatically.
